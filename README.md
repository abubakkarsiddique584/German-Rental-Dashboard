# 🏙️ German Rental Market Dashboard

An interactive full-stack data analytics application that explores and visualizes the German housing market. This project processes over 260,000 real-world rental listings to provide actionable insights for apartment seekers.

## 🚀 Key Features
- **Comprehensive Search:** Filter by city, budget, and minimum room count.
- **German Market Specifics:** Advanced filtering for Fitted Kitchens (EBK), Balconies, Gardens, and Lift access.
- **Interactive Visualization:** Dynamic Plotly Box Plots showing rent-per-square-meter distribution across major cities.
- **Real-Time Statistics:** Instant calculations for total listings, average market rent, and the priciest city in the database.
- **Responsive UI:** Optimized layout for laptop and desktop use with a professional SaaS-style sidebar.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Data Processing:** Pandas, NumPy
- **Frontend/Visualization:** Plotly.js, HTML5, CSS3 (Inter Typography)
